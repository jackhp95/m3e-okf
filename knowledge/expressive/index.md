# Expressive

What Material 3 Expressive (M3E) adds over baseline Material 3: physics-based motion, an expanded shape system with morphing, and an emphasized typography scale — grounded in a large research program. Start with what-changed, then read the concept you need.

## Concepts

| Concept | What it covers |
| --- | --- |
| [Applying expressive](/expressive/applying-expressive) | How to use motion, shape, and emphasis to guide attention without overwhelming a UI. |
| [Emphasized type](/expressive/emphasized-type) | An emphasized typography scale that pushes key lines forward in the hierarchy without resizing them. |
| [Motion physics](/expressive/motion-physics) | Springs replace fixed easing and duration, giving transitions natural momentum. |
| [Research rationale](/expressive/research-rationale) | The usability evidence behind M3E — 46 studies, 18k+ participants, and eye-tracking findings on how much faster people find key elements. |
| [Shape morphing](/expressive/shape-morphing) | An expanded shape library plus animated transforms that morph a control between shapes on interaction. |
| [What changed in Material 3 Expressive](/expressive/what-changed) | The three foundational shifts M3E makes over baseline Material 3 — motion, shape, and typography. |
