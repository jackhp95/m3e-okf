# Components

Neutral per-class concept files — anatomy, usage, and accessibility for each component class in technology-neutral terms. Each links to its tag-level, CEM-verified counterpart under /implementations/m3e-web/components.

## Concepts

| Concept | What it covers |
| --- | --- |
| [App bar](/components/app-bar) | A bar at the top or bottom of a screen holding titling, navigation, and key actions. |
| [Autocomplete](/components/autocomplete) | A text field that suggests and filters matching options as the user types. |
| [Avatar](/components/avatar) | A small graphic representing a person or entity, as an image, initials, or icon. |
| [Badge](/components/badge) | A small overlay marker that signals a count or a state on another element. |
| [Bottom sheet](/components/bottom-sheet) | A surface anchored to the bottom edge that presents supplementary content or actions. |
| [Breadcrumb](/components/breadcrumb) | A trail showing the user's location within a hierarchy, with links back to ancestor levels. |
| [Button group](/components/button-group) | A set of related buttons presented together as a connected unit. |
| [Button](/components/button) | A labeled control that performs an action, in emphasis variants from filled to text. |
| [Calendar](/components/calendar) | A month-grid view for browsing and selecting dates. |
| [Card](/components/card) | A contained surface grouping related content and actions about a single subject. |
| [Checkbox](/components/checkbox) | A control for selecting any number of options independently, including an indeterminate state. |
| [Chips](/components/chips) | Compact elements for input, filtering, selection, or triggering an action in context. |
| [Date picker](/components/date-picker) | A control for choosing a date or range, by calendar or by direct text entry. |
| [Dialog](/components/dialog) | A modal surface that interrupts for a decision or focused task, blocking the page behind it. |
| [Divider](/components/divider) | A thin line that separates content into groups without adding a heavier container. |
| [Expansion panel](/components/expansion-panel) | A titled section that expands to reveal its content and collapses to hide it. |
| [FAB menu](/components/fab-menu) | A floating action button that expands into a set of related primary actions. |
| [Floating action button](/components/fab) | A prominent, elevated control representing the single primary action of a screen. |
| [Icon button](/components/icon-button) | A compact control that performs an action using only an icon, common in toolbars and dense contexts. |
| [Icon](/components/icon) | A small symbolic graphic that represents an action, object, or concept. |
| [List](/components/list) | A vertical arrangement of related items in a consistent row format. |
| [Loading indicator](/components/loading-indicator) | A compact expressive indicator that signals a brief, ongoing wait. |
| [Menu](/components/menu) | A temporary overlay of choices or actions anchored to the control that opened it. |
| [Navigation bar](/components/navigation-bar) | A bottom bar of top-level destinations for compact-width screens. |
| [Navigation drawer](/components/navigation-drawer) | A side panel of destinations, permanent on wide screens or modal on demand, for larger destination sets. |
| [Navigation rail](/components/navigation-rail) | A vertical strip of top-level destinations along the leading edge, for medium-width screens. |
| [Paginator](/components/paginator) | A control for moving through a large data set one page at a time. |
| [Progress indicator](/components/progress-indicator) | A linear or circular indicator showing that a process is underway, determinate or indeterminate. |
| [Radio group](/components/radio-group) | A set of mutually exclusive options where exactly one can be selected. |
| [Segmented button](/components/segmented-button) | A compact row of connected buttons for choosing among a few related options, single- or multi-select. |
| [Select](/components/select) | A control that reveals a list of options and lets the user choose one, saving space over inline options. |
| [Slider](/components/slider) | A control for selecting a value or range along a continuous or stepped track. |
| [Snackbar](/components/snackbar) | A brief, non-blocking message shown at the edge of the screen, optionally with a single action. |
| [Split button](/components/split-button) | A button pairing a primary action with an attached menu of related choices. |
| [Stepper](/components/stepper) | A control that guides the user through an ordered sequence of steps, showing progress through them. |
| [Switch](/components/switch) | A toggle for turning a single setting on or off with immediate effect. |
| [Tabs](/components/tabs) | A row of peer views the user switches between, showing one panel at a time. |
| [Text field](/components/text-field) | A labeled input for entering text, with supporting and error text, in filled or outlined styles. |
| [Toolbar](/components/toolbar) | A grouped set of actions relevant to the current content, often floating near it. |
| [Tooltip](/components/tooltip) | A small transient label that identifies or explains an element on hover or focus. |
| [Tree](/components/tree) | A hierarchical list whose nodes expand and collapse to reveal nested children. |
