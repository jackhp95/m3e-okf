Icons are variable-font symbols controlled by four **axes**, so a single icon set
can be tuned rather than swapped for differently drawn versions.

## The four axes

- **Fill** — outlined versus filled.
- **Weight** — stroke thickness.
- **Grade** — fine adjustment to visual weight.
- **Optical size** — the size the glyph is optically tuned for.

## Choosing values

One icon style is kept across a screen so the set reads as a single family. A
filled state commonly signals a selected or active item while the outlined state
carries the rest state.
